
# Python Project Starter Kit

1. Install Python3.9.X 
    - https://www.python.org/downloads/release/python-3910/
2. pip install pipenv
3. SystemPropertiesAdvanced
    - PIPENV_VENV_IN_PROJECT = true
4. pipenv install
